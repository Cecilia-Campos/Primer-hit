Comando para git:

  1. Primer comando: git init (para inicializar git, una sola vez por proyecto)
  2. Segundo comando: git status (para chequear el estado de git en nuestro proyecto)
  3. Tercer comando: git add . (para trackear todos los archivos)
  4. Cuarto comando: git commit -m "Mensajito" (para hacer el paquetito de una versión, que podamos subir a la nube)
